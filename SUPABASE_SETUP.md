# Подключение Supabase

Для работы с реальной базой данных и авторизацией выполните следующие шаги:

1.  **Создайте проект Supabase**
    Перейдите на [supabase.com](https://supabase.com/) и создайте новый проект.

2.  **Настройте базу данных**
    В панели управления Supabase перейдите в **SQL Editor** и выполните скрипт из файла `supabase_schema.sql` (находится в корне проекта). Это создаст таблицу `programs` и настроит права доступа.

3.  **Получите ключи API**
    В настройках проекта (Project Settings -> API) скопируйте:
    *   Project URL
    *   anon public key

4.  **Настройте переменные окружения**
    Создайте файл `.env` в корне проекта (если его нет) и добавьте туда ключи:
    ```env
    VITE_SUPABASE_URL=ваша_ссылка_на_проект
    VITE_SUPABASE_ANON_KEY=ваш_anon_ключ
    ```

5.  **Перезапустите приложение**
    После добавления переменных окружения перезагрузите страницу. Приложение автоматически переключится с локального хранилища на Supabase.
